 Application Usage Tracker
A simple and effective application for tracking usage time of active applications on your Windows machine. This Python program provides real-time tracking, visual summaries, and total usage statistics for all the applications you use.

Features
Real-Time Tracking: Automatically tracks the time spent on each application in the foreground.
Pie Chart Visualization: Displays a pie chart of application usage for easy understanding.
Usage Summaries: Get detailed summaries of usage time for each application.
Most Used Category: Quickly find out which application you use the most.

The application will begin tracking the active application automatically.
Use the buttons to display:
App Usage Pie Chart
Most Used Category
Total Summary of usage time.
